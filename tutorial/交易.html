解锁所有交易物品


我们打开村民交易的 UI

只能看见村民当前等级交易的物品

实际上，更高等级的交易项目被隐藏了

这个界面文件是trade_2_screen.json
(pocket UI档案的交易项目也在这个json文件)

要做到样其实很简单
打开文件，复制下方代码替换整个文件。

"tier_stack_panel": { "bindings": [] }, "trade_toggle": { "$toggle_enabled_binding_type": "none" }

最后，保存并打开Minecraft，这样就能解锁所有交易项目了。
