蒙版剪辑

首先准备两张不同效果图片

我们首先把第一张图片设为背景
代码示例

"default_image": { "type": "image", "texture": "图片路径", "fill": true, "size": ["100%","100%"], "controls": [] }

在controls内添加亿点点控件：

"background": { "type": "image", "texture": "textures/ui/background", "layer": -100, "size": ["100%","100%"], "fill": true, "controls": [ { "line_mask": { "type": "panel", "size": ["100%", "100%"], "clips_children": true, "offset":["-70%",0], "controls": [ { "mask_image": { "type": "image", "texture": "textures/ui/new_background", "size": ["100%","100%"], "offset":["70%",0], "fill": true } } ] } } ] }

其中运用了clips_children和offset属性来实现蒙版剪辑：

属性作用
clips_children剪辑模板外多余部分。
offset改变蒙版剪辑的方向。

line_mask控件的offset为向左移动70%，表示保留无蒙版图片剩下70%的部分；
接着让mask_image控件向右移动70%，表示去除蒙版图片剩下70%的部分。

我们可以发现，这两者的的offset刚好相反，
这样才能形成蒙版剪辑的效果。
