控件常用属性

type 控件类型

值：字符串
定义这个控件的类型
字符串的值可能为

  • screen
  • panel
  • stack_panel
  • input_panel
  • label
  • image
  • button
  • toggle
  • dropdown
  • grid
  • slider
  • edit_box
  • custom
  • factory
  • scroll_view
  • scroll_track
  • scrollbar_box
  • selection_wheel

 color 控件颜色

值：数组 | 字符串
默认值：[ 1, 1, 1 ]
能使用RGB或列举的颜色改变控件颜色

.bb值作为字符串时..bb
可以填写以下的颜色
白色 white
淡灰色 silver
灰色 gray
黑色 black
红色 red
绿色 green
黄色 yellow
棕色 brown
黄绿色 greenyellow
淡蓝色 cyan
蓝色 blue
橙色 orange
...

.bb值作为数组时..bb
可以作为为RGB格式
数组的值必须为整数且范围在0到1之间

.bb如何将（255色）转换为0到1的数？..bb
R、G、B分别除于255
例如：[ 11, 45, 14 ] ÷ 255
≈ [ 0.0431, 0.1764, 0.0549 ] （保留4位小数）

enabled 是否启用控件

值：布尔值
默认值：true
为true时启用显示，为false时锁定控件
仅限于可互动类控件
锁定时，无法改变控件状态

visible 是否显示控件

值：布尔值
默认值：true
为true时控件显示，为false时隐藏控件
可与ignored配合使用

ignored 是否隐藏控件

值：布尔值
默认值：false
为true时隐藏显示，为false时显示控件
可与visible配合使用

.bb alpha ..bb .et 透明度 ..et

值：数字/浮点数
默认值：1
改变这个控件的透明度，但不影响子控件的透明度
浮点数的范围在0~1之间

.ep.bb propagate_alpha ..bb .et 传播透明度 ..et

值：数字/浮点数
默认值：1
改变子控件以及当前控件的透明度
浮点数的范围在0~1之间

..ep .ep.bb clips_children ..bb .et 剪辑子控件 ..et

值：布尔值
默认值：false
为true时，如果子控件超出当前控件的大小会被剪辑
除非子控件"allow_clipping": true

..ep .ep.bb allow_clipping ..bb .et 允许剪辑 ..et

值：布尔值
默认值：true
为true时，如果子控件超出某个剪辑控件的大小不会被剪辑

..ep .ep.bb clip_state_change_event ..bb .et 改变剪辑状态的事件 ..et

值：字符串
