type 控件类型

值：字符串
定义这个控件的类型
字符串的值可能为

  • screen
  • panel
  • stack_panel
  • input_panel
  • label
  • image
  • button
  • toggle
  • dropdown
  • grid
  • slider
  • edit_box
  • custom
  • factory
  • scroll_view
  • scroll_track
  • scrollbar_box
  • selection_wheel

size 控件大小

值：数组
默认值：[ "default", "default" ]
size分别为[ x, y ]
即用于定义X/Y轴大小

X/Y轴的值可以为

  • 像素
  • 默认
  • 百分比
  • 子元素百分比
  • 最大子元素百分比
  • 最大父元素百分比
  • 填充
  • Y轴百分比
  • X轴百分比
提示
1. 以下提到的父元素指嵌套*当前控件*的控件（即父控件的controls含有*当前控件*）
2. 以下提到的子元素指*当前控件*的嵌套控件（即*当前控件*的controls含有子控件）
> 像素
值: 数字 | 字符串
例如: [1,9] ["3px","5px"]
即使父空间多小，也影响不了当前控件的像素大小
通常不用带上"px"

> 默认
值: 字符串
保持控件原有的大小

> 百分比
值: 字符串
例如: ["100%","50%"]
即当前控件占用父控件的百分比

> 子元素百分比
值: 字符串
例如: ["100%c","50%c"]
即子元素加起来的大小占比
例如这个控件的两个子元素大小有["100%",2]["20%",2]
控件大小为["100%c","50%c"] => ["120%",2]

> 最大子元素百分比
值: 字符串
例如: ["100%cm",10]
即当前控件的子元素最大的大小占比
例如这个控件的两个子元素大小有["100%",8]["20%",9]
控件大小为["100%cm",2] => ["100%",2]

> 最大父元素百分比
值: 字符串
例如: ["100%sm",10]
即当前控件的父元素中的子元素最大的大小占比
例如这个控件的父元素大小是["100%c",8]，其中父元素的最大子元素大小是[180,6]
控件大小为["100%sm",2] => [180,2]

> 填充
值: 字符串
例如: ["fill",10]
即父元素的剩余大小(仅适用于stack_panel)
例如这个控件的父元素的剩余大小为[180,1]
控件大小为["fill",1] => [180,1]

> Y轴百分比
值: 字符串
例如: ["100%y",10]
即当前控件Y轴的百分比
例如这个控件大小为["50%y",100]
控件大小为["50%y",100] => [50,100]

> X轴百分比
值: 字符串
例如: ["100%","100x"]
即当前控件X轴的百分比
例如这个控件大小为["100%","100%x"]
控件大小为["100%","100%x"] => ["100%","100%"]

你不能这样做，会导致控件出现问题
示例:
["100%x","100%y"]
["100%y","100%x"]
["100%x","100%"]
["100%","100%y"]
["-100%","100%"]
(size不能为负数，负数一般用于offset)

你知道吗:
+ "size": [0,0]时可以将控件隐藏，但是仍然建议使用ignored隐藏控件
+ "size"可以使用加减符号，例如["100% + 2% - 9px","20%"]（像素大小在参与加减计算中必须加上px）

layer 显示的图层级

值：整数
默认值：0
能防止当前控件与其它控件重叠
层数越大控件将会把层数小的控件覆盖

enabled 是否启用控件

值：布尔值
默认值：true
为true时启用显示，为false时锁定控件
仅限于可互动类控件
锁定时，无法改变控件状态

visible 是否显示控件

值：布尔值
默认值：true
为true时控件显示，为false时隐藏控件
可与ignored配合使用

ignored 是否隐藏控件

值：布尔值
默认值：false
为true时隐藏显示，为false时显示控件
可与visible配合使用

offset 偏移量

值：数组/坐标
默认值：[0,0]
偏移是基于父元素的左上角，因此正方向为右下方(即负-y正+x，第四象限)
数组的值与size一致

anchor_from 开始锚点

值：字符串/列举
默认值：center
改变控件的开始(左上角)锚点
列举的值有：

top_lefttop_middletop_right
left_middlecenterright_middle
bottom_leftbottom_middlebottom_right

anchor_to 结束锚点

值：字符串/列举
默认值：center
改变控件的结束(右下角)锚点
列举的值有：

top_lefttop_middletop_right
left_middlecenterright_middle
bottom_leftbottom_middlebottom_right
