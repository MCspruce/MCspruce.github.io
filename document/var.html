继承

继承定义：变量继承到某个的控件的属性

在原版资源包UI文件夹中
里面存放一个叫_global_variables.json的文件
这是文件里面的内容

{ //////// Text colors //////// "$generic_button_text_color": [ 1.0, 1.0, 1.0 ], "$light_button_default_text_color": [ 0.3, 0.3, 0.3 ], "$light_button_hover_text_color": [ 1.0, 1.0, 1.0 ], "$light_button_pressed_text_color": [ 1.0, 1.0, 1.0 ], "$light_button_locked_text_color": [ 0.7, 0.7, 0.7 ], …还有很多 }

可以发现，代码都是"$变量": "值"

那么要如何在控件中使用变量呢？

"test_panel": { "type": "panel", "size": [ "100%", "100%" ], "controls": [ { "test_image": { "type": "image", "texture": "cube_ui/textures/common/icons/shortcut_setting_glyph_color", "size": [ 32, 32 ], "layer": 10 } }, { "text": { "type": "label", "size": [ "100%", "default" ], "text": "设置", "color": [ 0.8, 0.8, 0.8 ], "offset": [ 0, 32 ], "layer": 10 } } ] } 使用继承（变量）效果 ▼ "test_panel": { "type": "panel", "size": [ "100%", "100%" ], "$layer": 10, "controls": [ { "test_image": { "type": "image", "texture": "cube_ui/textures/common/icons/shortcut_setting_glyph_color", "size": [ 32, 32 ], "layer": "$layer" } }, { "text": { "type": "label", "size": [ "100%", "default" ], "text": "设置", "color": [ 0.8, 0.8, 0.8 ], "offset": [ 0, 32 ], "layer": "$layer" } } ] }

可以看到继承需要在控件属性的值修改为"$变量名称"

例如 "layer": "$变量"

接着给变量赋值

例如 "$变量": 5

提示
如果变量没有赋值，变量的值默认设为默认值

注意事项

错误示范

错误原因：继承的赋予的值一致，并且text变量不支持数组

"test": { "$a": [ 0.1, 1, 1 ], "color": "$a", "text": "$a" } 正确示范 "test": { "$a": [ 0.1, 1, 1 ], "$b": "文字", "color": "$a", "text": "$b" }

变量默认值

在RP/ui/start_screen.json文件的start_screen_content控件

"start_screen_content": { "type": "panel", "$achievements_ignored|default": false, "$feedback_ignored|default": false, "$store_ignored|default": false, "$is_toolbox_ignored|default": false // … 还有更多 }

面板中有个变量

"$achievements_ignored|default": false 这是什么意思呢?

你可以在变量赋值时，在变量后面填加|default将变量设为默认值

这个意思是说$achievements_ignored的默认值为false

下面举个例子

"test": { "$test": "a", "$test|default": "b", "text": "$test" }

这样情况下 text的值为"b"

因为"$test": "a"覆盖了"$test|default": "b"
这样可以说明
在同一个变量下，没带|default的变量会覆盖原有的默认变量

全局变量

在前面的_global_variables.json文件中有很多继承变量

当然赋值也可以在这个文件里面进行，这样所有文件都可以使用这个变量

赋值示范

{ "$text": "a" // 可以添加多个变量 }
