文本控件类型为label

接下来，我们给文本添加一些属性


常用属性
.tip

提示：
size等常用属性被整理至另外一个文件

点击查看

..tip .ep.bb text ..bb .et 文字 ..et

值：字符串 .hh定义文本控件显示的文字 .hh可以使用 § 符号进行改变颜色 .hh“\n”可以使文本换行 .hh同时可以使用.lang文件翻译来改变文本语言

..ep .ep.bb color ..bb .et 文本颜色 ..et

值：数组 | 字符串 .hh默认值：[ 1, 1, 1 ] .hh能使用RGB或列举的颜色改变字体颜色

.bb值作为字符串时..bb .hh可以填写以下的颜色 .hh白色 white .hh淡灰色 silver .hh灰色 gray .hh黑色 black .hh红色 red .hh绿色 green .hh黄色 yellow .hh棕色 brown .hh黄绿色 greenyellow .hh淡蓝色 cyan .hh蓝色 blue .hh橙色 orange .hh... .hh .hh.bb值作为数组时..bb .hh可以为RGB .hh数组的值必须为整数且范围在0到1之间 .hh.bb如何将（255色）转换为0到1的数？..bb .hhR、G、B分别除于255 .hh例如：[ 11, 45, 14 ] ÷ 255 .hh≈ [ 0.0431, 0.1764, 0.0549 ] （保留4位小数）

..ep
其他属性
.ep.bb locked_color ..bb .et 锁定时的文本颜色 ..et

值：数组 | 字符串 .hh与"color"的值相同，不再介绍 .hh需要父控件"enabled": false以应用"locked_color"为文本颜色

..ep .ep.bb locked_alpha ..bb .et 锁定时的文本透明度 ..et

值：数组 .hh与"alpha"的值相同，不再介绍 .hh需要父控件"enabled": false以应用"locked_alpha"为文本透明度

..ep .ep.bb shadow ..bb .et 影子 ..et

值：布尔值 .hh默认值：false .hh启用时，文字右下方会有影子渲染 .hh影子样式不可自定义

..ep .ep.bb hide_hyphen ..bb .et 隐藏由分词引起的自动换行 ..et

值：布尔值 .hh默认值：false

..ep .ep.bb notify_on_ellipses ..bb .et 去掉省略号时要改变的控件 ..et

值：数组 例如："notify_on_ellipses": ["控件名"]

..ep .ep.bb enable_profanity_filter ..bb .et 启用敏感词 ..et

值：布尔值 .hh默认值：false .hh为true时启用，启用后控件中的敏感词作为“***”显示 .hh目前Minecraft暂不支持简体中文的敏感词

..ep .ep.bb font_size ..bb .et 字体大小 ..et

值：字符串/列举 .hh默认值：normal .hh用于改变字体大小 .hh值可以是 .hhsmall - 小 .hhnormal - 正常 .hhlarge - 大 .hhextra_large - 更大

..ep .ep.bb font_scale_factor ..bb .et 字体缩放比例 ..et

值：数字/浮点数 .hh默认值：1.0 .hh用于改变字体缩放比例 .hh以font_size为基础缩放

..ep .ep.bb localize ..bb .et 本地化 ..et

值：布尔值 .hh默认值：false .hh为true时，不能被.lang文件翻译

..ep .ep.bb line_padding ..bb .et 行间距 ..et

值：数字 .hh改变文字间的行间距

..ep .ep.bb font_type ..bb .et 字体 ..et

值：字符串/列举 .hh改变文字的字体 .hh如果text包含“中文”则强制unicode .hh网易中国版改变font_type会引起字体显示问题 .hh可能的值： .hhdefault - 默认 .hhrune - 附魔字体 .hhMinecraftTen - MinecraftTen .hhunicode - unicode .hhsmooth - 平滑字体 .hh提示：添加第三方字体会引起崩溃

..ep .ep.bb backup_font_type ..bb .et 备份字体 ..et

值：字符串/列举 .hh不起作用时，改变文字的备份字体 .hh如果text包含“中文”则强制unicode .hh网易中国版改变font_type会引起字体显示问题 .hh可能的值： .hhdefault - 默认 .hhrune - 附魔字体 .hhMinecraftTen - MinecraftTen .hhunicode - unicode .hhsmooth - 平滑字体 .hh提示：添加第三方字体会引起崩溃

..ep .ep.bb text_alignment ..bb .et 文字对齐 ..et

值：字符串/列举 .hh默认情况下，对齐方向以anchor_from，anchor_to为准 .hh可能的值： .hhcenter，left，right

..ep
代码示例
简单的文本控件 "label": { "type": "label", "text": "test" } 强制unicode的文本控件 "label": { "type": "label", "text": "test", "font_type": "unicode" } 使用notify_on_ellipses的文本控件 "label": { "type": "label", "text": "test", "notify_on_ellipses": [ "控件" ] }, "控件": { "type": "panel" ... }
