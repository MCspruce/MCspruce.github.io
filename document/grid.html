网格

属性

属性属性名值默认值描述
grid_dimensions网格尺寸矢量[row, col]网格尺寸。既网格的行数和列数。
maximum_grid_items最大网格物品整数限制网格最大物品数量，默认没有限制。
grid_dimension_binding网格尺寸绑定字符串网格尺寸的绑定。
grid_rescaling_type网格重新缩放类型字符串：列举none网格重新缩放类型。
列举的值可以是：
none 无
vertical 垂直
horizontal 水平
grid_fill_direction网格填充方向字符串：列举none网格填充方向。
列举的值可以是：
none 无
vertical 垂直
horizontal 水平
grid_item_template网格物品模板字符串网格物品模板。
属性的值格式为"namespace.controls_name"
