JSON 语法

要了解 json UI，那就必须了解json(JavaScript Object Notation)语言。

如果想深入了解JSON，可以前往www.w3school.com.cn/js/js_json_syntax.asp

JSON 语法

JSON支持以下几种基本数据类型：

字符串: 由双引号 "" 或单引号 '' 包围的文本，使用 \n 代表换行。

数字: 表示一个数值。

布尔值: 表示真假值，true 为真，false 为假。

对象: 包含键值对的无序集合。

数组: 有序的值的列表。

Null: 表示空值。

JSON 压缩和格式化

JSON 压缩：去除空格、换行等多余的字符来减小 JSON 文件大小。

JSON 格式化：有缩进和换行，容易阅读。

JSON 注释

注释后的代码更容易了解这部分是什么内容，并且不会被解析。

JSON 单行注释：使用双斜线//注释一行代码。

JSON 多行注释：使用/* */将多行代码包围，其中的内容为注释。

JSON 转义和 Unicode

在字符串中我们可以使用转义，来代替某些特殊字符。
例如 \" 为双引号， \\ 为反斜杠。
此外，Unicode 可以通过 \u 后接四位十六进制数字来表示。
