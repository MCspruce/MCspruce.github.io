    了解MC存档路径

    Windows：C:\Users\USERNAME\AppData\Local\Packages\Microsoft.MinecraftUWP_8wekyb3d8bbwe\LocalState\games\com.mojang
    Android/鸿蒙：
    （1.17或1.17以前）games/com.mojang
    （1.18或1.18以后）Android/data/com.mojang.minecraftpe/files/games/com.mojang
    IOS/IPadOS：我的xxx/Minecraft/games/com.mojang
    ## manifest.json 这个文件用于被Minecraft识别，必须存放在 UI包的根目录 文件示例（UUID自己生成）： ```json { // 支持1.12或以下版本填1，否则填2（建议填"1" "format_version": 2, "header": { "name": "资源包名字", "description": "资源包描述", // UUID "uuid": "", // 资源包版本 "version": [1,0,0] }, "modules": [ { "type": "resources", // 资源包版本 "version": [1,0,0], // 这里的uuid不要与上方一致 "uuid": "" } ] } ```
