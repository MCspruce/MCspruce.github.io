对于新手来说，不知道如何添加控件？ hh这篇文章主要讲解如何添加控件。

那么要如何添加控件呢？

RP/ui/start_screen.json { "namespace": "start", "test": { "type": "label", "text": "hello world" } }

像这样，在json根添加新json对象(即控件)即可。

但是你不能像下面的代码这么做！

{
  "namespace": "start",
  
  // 这样做是错误的，请不要模仿！
  {
    "test": {
      "type": "label",
      "text": "hello world"
    }
  }
}

添加一个控件就是如此简单，那么如何让控件在屏幕上显示呢？

RP/ui/start_screen.json { "namespace": "start", "test": { "type": "label", "text": "hello world" }, "start_screen_content": { "controls": [ {"test@test": {}}, {"main_buttons_and_title_panel@start.main_buttons_stack_panel": {}}, { "online_button_stack": { "type": "stack_panel", "orientation": "vertical", "anchor_to": "bottom_left", "anchor_from": "bottom_left", "size": [ "50% - 75px", "100%c" ], "offset": [ 0, "-10%" ], "controls": [ {"gathering_panel@start.gathering_panel": {}}, {"upper_online_buttons_panel@start.upper_online_buttons_panel": {}}, {"lower_online_buttons_panel@start.lower_online_buttons_panel": {}} ] } }, {"skin_panel@start.skin_panel": { "layer": 1 }}, {"text_panel@start.text_panel": {}} } ] } }

在代码中，删掉了一些无关此教程的代码。

你也可以像上面的代码这样做，因为属性已经在原版代码中声明，我们要做的就是重新声明"controls"！重新声明"controls"可以覆盖原版的"controls"以达到显示控件的效果。

那么要如何找到每个屏幕文件的XXX_screen_content？

很简单！每个代码编辑器都有个搜索功能。在搜索框输入并搜索“_screen_content”即可
