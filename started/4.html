创建控件

对于新手来说，不知道如何添加控件？ hh这篇文章主要讲解如何添加控件。

那么要如何添加控件呢？

{ "namespace": "start", "test": { "type": "label", "text": "hello world" } }

像这样，在json根添加新json对象(即控件)即可。

添加一个控件就是如此简单，那么如何让控件在屏幕上显示呢？

{ "test": { "type": "label", "text": "hello world" }, "start_screen_content": { "modifications": [ { "array_name": "controls", "operation": "insert_back", "value": [ {"test@test": {}} ] } ] } }

在代码中，删掉了一些无关的代码。

你也可以像上面的代码这样做。我们使用了modifications属性添加控件，这样可以在添加控件的同时尽可能减少错误。

那么要如何找到每个屏幕文件的XXX_screen_content？

很简单！每个代码编辑器都有个搜索功能。在搜索框输入并搜索“_screen_content”即可（仅对部分屏幕有效）
